## Sobre o Projeto

Aplicativo mobile para mapear, cadastrar e dar visibilidade aos agentes culturais do município de Confins/MG. Este repositório contém a **documentação de planejamento** (cronograma, escopo, arquitetura) — o desenvolvimento do código está em fase de planejamento.

---

## Contexto

Confins/MG não possui um mapeamento digital dos seus agentes e espaços culturais. Este app visa resolver isso criando um diretório acessível via celular, conectando artistas, produtores e público.

## Funcionalidades Previstas

- Cadastro de agentes culturais (artistas, grupos, espaços)
- Mapa interativo com geolocalização
- Busca e filtros por área artística
- Perfil público dos agentes
- Agenda de eventos culturais locais
